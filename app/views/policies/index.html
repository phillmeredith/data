Data Ethics Framework

The NHSBSA Data Ethics Framework provides guidance on the ethical use of data within the organization. It helps ensure that data is used in a way that is:

  • Transparent - Clear about how and why data is being used
  • Accountable - Taking responsibility for data use and its impacts
  • Fair - Ensuring data use does not discriminate or reinforce biases
  • Secure - Protecting data from unauthorized access or misuse
  • Proportionate - Using only the data necessary for the intended purpose
