Request Data

Use this service to request data from the NHS Business Services Authority.

You can use this service to:

  • Request specific datasets
  • Define custom data requirements
  • Specify output formats and delivery methods
  • Track the status of your requests
Information:

All data requests are subject to approval and data governance requirements. Sensitive or identifiable data may require additional approvals.

Before you start

You will need:

  • Your NHS email address
  • Information about the data you require
  • Details of how you intend to use the data
  • Any specific time periods or geographical areas you're interested in

The request process takes approximately 5-10 minutes to complete.
