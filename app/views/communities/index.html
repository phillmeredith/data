Our Communities

Connect with data communities and find support for your data needs. This section provides information on how to connect with other data users, access training, and get help with using NHSBSA data.

Data User Community

Join our community of data users across the NHS

The NHSBSA Data User Community brings together data analysts, managers, and users from across the NHS to share knowledge, best practices, and support each other in using NHSBSA data effectively.

Benefits of joining:

  • Connect with peers facing similar challenges
  • Share knowledge and best practices
  • Get early access to new datasets and features
  • Influence the development of the Data Hub
  • Participate in user research and testing
Join the community

Support and Help

Get help with using NHSBSA data and systems

Our support team is available to help you with any questions or issues you may have when using NHSBSA data or systems.

Contact methods:

  • Email: datahub@nhsbsa.nhs.uk
  • Phone: 0300 330 1348 (Monday to Friday, 9am to 5pm)
  • Online chat: Available through the Data Hub portal

Support offered:

  • Help with accessing systems and datasets
  • Guidance on using dashboards and reports
  • Assistance with data requests
  • Technical support for data-related issues
  • Help with interpreting data and metadata
