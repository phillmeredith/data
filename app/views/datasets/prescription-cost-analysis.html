Description

The Prescription Cost Analysis (PCA) dataset provides detailed information on prescriptions dispensed in the community in England. It includes data on the number of items dispensed, the net ingredient cost, and the actual cost of prescriptions.

This dataset is useful for:

  • Monitoring prescribing patterns and trends
  • Identifying cost-saving opportunities
  • Supporting formulary development and review
  • Informing commissioning decisions
  • Supporting research into prescribing behaviors
